Conditional Statements in C

The syntax of the if statement in C is:

if (test expression) {
    // code
}

The if statement evaluates the test expression inside the parenthesis (). If the test expression is evaluated to true, statements inside the body of if are executed. If the test expression is evaluated to false, statements inside the body of if are not executed.

The if statement comes with an optional else clause. If the else clause is added then if the text expression in the if statement is false then the else statement will be executed. The syntax of the if...else statement is:

if (test expression) {
    // run code if test expression is true
} else {
    // run code if test expression is false
}

So using this the conditional statements if and else if can control the program flow to execute certain code blocks if certain conditions are satisfied. This can be useful not only for executing algorithms but for handlding user input to control the program. For example, if the user types "quit" we may want to close the program, or if the user input invalid input like a letter instead of a number we can offer validation and let the user know they made a mistake. Conditional Statement can take our programs to the next level.

if...else Ladder

The if...else statement executes two different codes depending upon whether the test expression is true or false. Sometimes, a choice has to be made from more than 2 possibilities. The if...else ladder allows you to check between multiple test expressions and execute different statements. It looks like so:

if (test expression1) {
    // statement(s)
}
else if(test expression2) {
    // statement(s)
}
else if (test expression3) {
    // statement(s)
}
.
.
.
else {
    // statement(s)
}

Here if the first test expression is not met the program will check the second text expression this will continue until a test expression is evaluated as true and that code block executed. If all of the else if test expression are false then the optional else statement is executed.

Note just like how you can have nested loop statements in C you can also have nested conditionals statements. This means you can place multiple if statements in a if statement. Check out the example below:

1 #include
2
3 int main() {
4     int number1, number2;
5     printf("Enter two integers: ");
6     scanf("%d %d", &number1, &number2);
7
8     if (number1 >= number2) {
9        if (number1 == number2) {
10            printf("Result: %d = %d",number1,number2);
11         } else {
12            printf("Result: %d > %d", number1, number2);
13         }
14     } else {
15         printf("Result: %d < %d",number1, number2);
16     }
17
18     return 0;
19 }

Cool right! So now we can check one condition in one if statement and another condition in the next if statement. This can be useful in assembly line logic, or cases where we do not want to run a computational test expression when we can run something simpler to reduce the number times we run the expensive check.

The switch statement, an alternative to the if...else Ladder

If you are running multiple test expressions with the same variable you may find the switch statement a great alternative to the if...else Ladder. A switch is a conditional statement which allows a variable to be tested for equality against a list of values. Each value is called a case, and the variable being switched on is checked for each switch case. It has the following syntax in C:

switch(expression) {
    case constant-expression:
        statement(s);
        break; /* optional */

    case constant-expression:
        statement(s);
        break; /* optional */

    /* you can have any number of case statements */
    default: /* Optional */
        statement(s);
}


The expression used in a switch statement must have an integral or enumerated type. You can have any number of case statements within a switch. Each case is followed by the value to be compared to and a colon.

The constant-expression for a case must be the same data type as the variable in the switch, and it must be a constant or a literal.

When the variable being switched on is equal to a case, the statements following that case will execute until a break statement is reached. When a break statement is reached, the switch terminates, and the flow of control jumps to the next line following the switch statement. Not every case needs to contain a break. If no break appears, the flow of control will fall through to subsequent cases until a break is reached.

A switch statement can have an optional default case, which must appear at the end of the switch. The default case can be used for performing a task when none of the cases is true. No break is needed in the default case.
