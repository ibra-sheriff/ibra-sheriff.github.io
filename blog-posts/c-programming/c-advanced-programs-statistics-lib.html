A basic Statistics Library in C

The challenge is to create a Statistics Library for integer data that can be used to compute common statistic measures like:

  • Average(Mean)
  • Median
  • Standard Deviation
  • Range

Given integer data in an array can you build a library to compute such statistics?

Requirements

In this section we will discuss the required functions. We need functions that take an integer array and the number of items in the array as parameters. We then use the array of integers to compute the required statistic, view the required declaration below:

int generate_random_number(int upper_limit, int lower_limit);
int* generate_random_array(int arr_size, int upper_limit, int lower_limit);
int array_sum(int values[], int n);
double array_average(int values[], int n);
double array_variance(int values[], int n);
double array_standard_deviation(int values[], int n);
double array_median(int values[], int n);
double array_range(int values[], int n);
int max_in_array(int values[], int n);
int min_in_array(int values[], int n);
void swap(int i, int j, int values[]);
void sort(int values[], int n);
void sort_desc(int values[], int n);
void print_array(int values[], int n);
The generate_random_number function needs to generate a random number within the given upper and lower limits.

The generate_random_array function is used to generate an array of \(n\) integers. The integers are random numbers between the given limits.

The array_sum function is used to compute the sum of all the values in an array.

The array_average function is used to compute the average(mean) of the integers in the given array.

The array_variance function is used to compute the variance of the integers in the given array.

The array_standard_deviation function is used to compute the standard deviation of the integers in the given array.

The array_median function is used to compute the median of the integers in the given array.

The array_range function is used to compute the range of the integers in the given array.

The max_in_array function is used to obtain the biggest value in the array.

The min_in_array function is used to obtain the smallest value in the array.

The swap function is used to swap two values in a given array e.g. the value at position 1 is replaced with the number at position 2 and the value that was at position 1 is put at position 2.

The sort function sorts the given array in ascending order i.e. from the smallest number to the biggest number.

The sort_desc function sorts the given array in descending order i.e. from the biggest number to the smallest number.
