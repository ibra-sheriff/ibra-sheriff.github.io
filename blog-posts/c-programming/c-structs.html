Grouping data in C using Structs

Structs can be thought of as containers for a group of values. The idea is that the programmer would like to store these values together as they relate to the same entity. For example, think of a student, we can have the following data:

  • The student's first name
  • The student's first surname
  • The student's mark for the exam
  • The student's mark for the project

By using a struct we can easily keep track of the date relating to one student even for all the fields above. Things get mess when you think about storing this information in an array I mean the first name and surname are arrays of characteters already so you end up with an array of arrays! Just for the names! How about the marks? One array for the exam marks and another array for the project marks? I guess it could work but it will be difficult to program in this manner, it would be better to use an array of structs where each student's information is kept in a struct. This way we just access position \(i\) in the array and can get the student's first name, surname, exam mark and project mark.

It's important to note that structs are just data containers, so C stores the data for a specific entity in the same place, we cannot assign functions to this data as we do with Object Oriented Programming in Java.

Student Data Program

We can define a struct to hold like so:

1 struct student {
2 char* first_name;
3 char* surname;
4 int exam_mark;
5 int project_mark;
6 };

We use the keyword struct and then enter the fields of the struct in the body of the struct. Here we put the type of the field e.g. char and the name of the field. Then we can then create a data entry for a student like so:

struct student student1 = {"Ibra", "Sheriff", 100, 100};

We have loaded the data for the student Ibra Sheriff, we can then access the fields use a dot e.g. the exam mark student1.exam_mark, for reading the value or changing it(e.g. student1.exam_mark = 98). Just like that we are packing our student in an organised manner using structs!

Now you may have noticed that we have to use 3 words to create a struct instead of two with other variables, this can be avoided by use of the keyword typedef when defining the struct as seen below:

1 typedef struct student {
2 char* first_name;
3 char* surname;
4 int exam_mark;
5 int project_mark;
6 }student;

Now can create a new struct like so:

student student1 = {"Ibra", "Sheriff", 100, 100};

Here typedef has allowed us to rename struct student to student throughout the program, much neater! Accessing is as before.
