Solving basic linear programming problems using the Simplex Algorithm

We have been given the following linear programming problem:

        max \(3x_1 + x_2 + 2x_3 \)

such that:

        \(x_1 + x_2 + 3x_3 \le 30 \)
        \(2x_1 + 2x_2 + 5x_3 \le 24 \)
        \(4x_1 + x_2 + 2x_3 \le 36 \)
        \(x_1, x_2, x_3 \ge 0 \)

First write it in Canonical form:

        \(z = 3x_1 + x_2 + 2x_3 \)
        \(x_1 + x_2 + 3x_3 + s_1 = 30 \)
        \(2x_1 + 2x_2 + 5x_3 + s_2 = 24 \)
        \(4x_1 + x_2 + 2x_3 + s_3 = 36 \)
        \(x_1, x_2, x_3 \ge 0 \)

First we fill in the values into the tableu. Then we calculate g where \(g_i = \sum_{n=1}^{3} bv_n \times col_{in} \)

Here \(bv_n\) is the \(n\)th value in the basis so that the moment: \(bv_1 = s_1 \), \(bv_2 = s_2 \) and \(bv_3 = s_3 \).
\(col_{in} \) is the \(n\)th item in the \(i\)th column.

\(z_i\) is simply calculated by substracting the number at the top of the column from \(g_i\), for example \(z_1 = g_1 - 3\), \(z_2 = g_2 - 1\), and \(z_3 = g_3 - 2\).

3 1 2 0 0 0 - -
Basis \(x_1 \) \(x_2 \) \(x_3 \) \(s_1 \) \(s_2 \) \(s_3 \) RHS Ratio Test
0 \(s_1 \) 1 1 3 1 0 0 30 \(\frac{30}{1} = 30\)
0 \(s_2 \) 2 2 5 0 1 0 24 \(\frac{24}{2} = 12\)
0 \(s_3 \) 4 1 2 0 0 1 36 \(\frac{36}{4} = 9\) ←
g 0 0 0 0 0 0 0
z -3 -1 -2 0 0 0


After selecting the smallest negative value we apply use that column to to do the Ratio Test. The smallest test aligns with our pivot row.
Now use the third row as the pivot row and apply elementary row operations as in Gauss-Jordan Elimination to make the 4 a 1 and all the other values in the column 0.

3 1 2 0 0 0 - -
Basis \(x_1 \) \(x_2 \) \(x_3 \) \(s_1 \) \(s_2 \) \(s_3 \) RHS Ratio Test
0 \(s_1 \) 0 \(\frac{3}{4} \) \(\frac{5}{2} \) 1 0 \(-\frac{1}{4} \) 21 \(21 \div \frac{5}{2} = 8.4\)
0 \(s_2 \) 0 \(\frac{3}{2} \) 4 0 1 \(-\frac{1}{2} \) 6 \(\frac{6}{4} = 1.5\) ←
3 \(x_1 \) 1 \(\frac{1}{4} \) \(\frac{1}{2} \) 0 0 \(\frac{1}{4} \) 9 \(9 \div \frac{1}{2} = 18\)
g 3 \(\frac{3}{4} \) \(\frac{3}{2} \) 0 0 \(\frac{3}{4} \) 27
z 0 \(-\frac{1}{4} \) \(-\frac{1}{2} \) 0 0 \(\frac{1}{4} \)

Rinse and repeat the steps as we still have a negative \(z_i \).

Basis \(x_1 \) \(x_2 \) \(x_3 \) \(s_1 \) \(s_2 \) \(s_3 \) RHS Ratio Test
0 \(s_1 \) 0 \(-\frac{3}{16} \) 0 1 \(-\frac{5}{8} \) \(\frac{1}{16} \) \(\frac{69}{4} \) \(\frac{69}{4} \div -\frac{3}{16} = X \)
2 \(x_3 \) 0 \(\frac{3}{8} \) 1 0 \(\frac{1}{4} \) -\(\frac{1}{8} \) \(\frac{3}{2} \) \(\frac{3}{2} \div \frac{3}{8} = 4\) ←
3 \(x_1 \) 1 \(\frac{1}{16} \) 0 0 \(-\frac{1}{8} \) \(\frac{5}{16} \) \(\frac{33}{4} \) \(\frac{33}{4} \div \frac{1}{16} = 132 \)
g 3 \(\frac{15}{16} \) 2 0 \(\frac{1}{8} \) \(\frac{11}{16} \) \(\frac{111}{14} \)
z 0 \(-\frac{1}{16} \) 0 0 \(\frac{1}{8} \) \(\frac{11}{16} \)

Now for the final tableau!

Basis \(x_1 \) \(x_2 \) \(x_3 \) \(s_1 \) \(s_2 \) \(s_3 \) RHS Ratio Test
0 \(s_1 \) 0 0 \(\frac{1}{2} \) 1 \(-\frac{1}{2} \) 0 18
1 \(x_2 \) 0 1 \(\frac{8}{3} \) 0 \(\frac{2}{3} \) \(-\frac{1}{3} \) 4
3 \(x_1 \) 1 0 \(-\frac{1}{6} \) 0 \(-\frac{1}{6} \) \(\frac{1}{3} \) 8
g 3 1 \(\frac{13}{6} \) 0 \(\frac{1}{6} \) \(\frac{2}{3} \) 28
z 0 0 \(\frac{1}{6} \) 0 \(\frac{1}{6} \) \(\frac{2}{3} \)

Hence we have the optimal solution of 28 for the objective function and \(s_1 = 18, x_1 = 8, x_2 = 4, x_3 = s_1 = s_2 = 0 \).

It's important to note that we can apply our Simplex algorithm only for maximising problems, so if you have a minimising simply start by negating the objective function and then apply the algorithm, for example:

min \(w = 2x_1 + 3x_2 + 4x_3 \) is the same as max \(w = -2x_1 - 3x_2 - 4x_3 \). Just remember to multiply your final answer \(w\) for the objective function by minus 1.

Steps to follow

(1) Select the column of interest by selecting the column with the smallest negative \(z_i\).
(2) Use the column of interest to run the Ratio test using the RHS. Note Ratio test that are negative or where the denominator is 0 are ignored as they are invalid.
(3) The pivot row is the row with the smallest Ratio test value. The pivot point is the intersection of the column of interest and the row with the smallest Ratio test value.
(4) Divide or multiply to make the pivot point 1 in the next tableau and then apply elementary row operations using the new values of the pivot value to make the values in the same column as the pivot point 0.
(5) If all \(z_i \ge 0\) then we have found the optimal solution. If there is no valid Ratio test then the problem has no feasible solution, otherwise we do another iteration.

For more checkout Operations Research Applications and Algorithms by Wayne L. Winston(2003) page 140 section 4.5. Note that Wayne using a different tableau format so \(g_i \) and \(z_i \) will likely be in a different place but the theory is exactly the same.
